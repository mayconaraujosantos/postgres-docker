## Postgres with Docker and Docker compose a step-by-step guide for beginners


## Step 1: copy .env.local to .env
```console
$ make env-file:
```
 
## Step 2: create  the container
```console
$ make up 
```

## Step 3: remove container  
```console
$ make down
```
